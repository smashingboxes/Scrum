Main Description

Scrum does not prescribe how the Product Backlog is prioritized.  Some techniques are:

  • Kano Analysis
  • Theme Screening
  • Theme Scoring
  • Relative Weighting

For more information see Prioritizing the Product Backlog by Mike Cohn.
